# Data Quality Checklist

## Core Dimensions
- Accuracy: Matches source data
- Completeness: No missing required fields
- Consistency: Standard formats applied
- Validity: Within acceptable ranges
- Uniqueness: No duplicate records
- Timeliness: Entered and updated promptly

## Validation Controls
- Required field enforcement
- Format validation (dates, emails, IDs)
- Duplicate detection
- Periodic audit review

## Common Risks
- Manual entry errors
- Duplicate records
- Outdated information
- Inconsistent formatting
